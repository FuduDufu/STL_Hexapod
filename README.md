# TCA-CS-5-Z.Dulguun Hexapod Robot 3D Parts

## Төслийн танилцуулга

Энэхүү репозиторид ойн зулзаган модны хяналтын системийн хүрээнд ашиглагдсан Hexapod роботын 3D хэвлэх загвар файлууд агуулагдана.

Төслийн зорилго нь ойн орчинд зулзаган мод илрүүлэх, тоолох, байршлыг тогтоох зориулалттай хөдөлгөөнт роботын платформ боловсруулах явдал юм. Энэхүү зорилгын хүрээнд зургаан хөлт (Hexapod) роботыг сонгон ашигласан бөгөөд уг робот нь бартаат зам, ойн хөрс зэрэг тэгш бус орчинд тогтвортой хөдөлгөөн хийх давуу талтай.

Энэхүү репозиторид роботын их бие, хөл, серво моторын бэхэлгээ, толгой болон бусад механик эд ангиудын 3D хэвлэх файлууд хадгалагдсан болно.


---

* PLA
* PLA+
* PETG

### Давхаргын өндөр (Layer Height)

```text
0.2 mm
```

### Infill

```text
20% - 40%
```

### Support

Зарим эд ангийг хэвлэхэд support ашиглах шаардлагатай байж болно.

## Эх сурвалж ба талархал

Энэхүү төслийн Hexapod роботын механик хийцийг боловсруулах явцад нээлттэй эхийн роботын төслүүдийг судалж ашигласан. Ялангуяа MakeYourPet төслийн Hexapod роботын загвар нь механик бүтэц болон угсралтын шийдлийг боловсруулахад чухал жишиг болсон.

Анхны төслийг дараах холбоосоос үзэх боломжтой.

https://github.com/MakeYourPet/hexapod

Энэхүү репозиторид байрлах загварууд нь судалгааны ажлын шаардлагад нийцүүлэн өөрчлөн боловсруулсан болон нэмэлт эд ангиудыг агуулж болно.
